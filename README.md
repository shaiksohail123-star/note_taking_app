## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/shaiksohail123-star/note_taking_app.git
```

2. Navigate to the project folder:

```bash
cd note_taking_app
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python app.py
```
